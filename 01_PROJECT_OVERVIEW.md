# Kiru - Project Overview

## App Name
**Kiru** (Destination Outfit Recommender)**

## Tagline
*AI-Powered Travel Stylist - Pack Smarter, Dress Better.

## Mission
Kiru revolutionizes travel packing by combining AI styling with your personal wardrobe, weather data, and destination insights to create perfect outfit recommendations for every trip.

## Vision
To become the global go-to app for travelers seeking personalized, culturally-sensitive, and sustainable travel fashion - making packing effortless and stylish for everyone.

## Core Values
1. **Personalization** - Every recommendation is tailored to YOU
2. **Simplicity** - Intuitive design that saves time
3. **Global** - Built for travelers worldwide
4. **Sustainability** - Promote using what you own first

---

## Project Goals (MVP Phase 1 (Core MVP)
- [ ] User profiles with style preferences
- [ ] Virtual wardrobe management
- [ ] Basic AI outfit recommendations
- [ ] Weather integration
- [ ] Trip planning with day-by-day outfits
- [ ] Packing list generator

## Phase 2 (Enhanced Features)
- [ ] AI chat stylist
- [ ] Outfit image generation
- [ ] AR try-on
- [ ] Social features

## Phase 3 (Premium Features)
- [ ] Advanced AI learning
- [ ] Full social community
- [ ] Creator marketplace
