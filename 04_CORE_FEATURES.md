# Kiru - Core Features Documentation

## Phase 1 - MVP Features

---

### 1. Onboarding & Profile
#### 1.1 Style Quiz
- **Swipe-based interface** (like Tinder)
- 20-30 outfit cards to swipe left/right
- Categories: Casual, Formal, Beach, Ethnic, Business, Streetwear
- Instant taste profile building

#### 1.2 Body Profile
- **Photo-based measurement** (AI estimates proportions)
- Manual input option
- Body type selection: Hourglass, Pear, Apple, Rectangle, Athletic
- Height, weight, typical sizes

#### 1.3 Skin Tone Detection
- Selfie upload for auto-detection
- Warm/Cool/Neutral classification
- Color palette recommendations based on skin tone

#### 1.4 Multiple Profiles
- Support for family members
- Pack for kids, partner, friends

---

### 2. Virtual Wardrobe
#### 2.1 Add Items
- **Camera scan** - point and shoot
- Photo upload from gallery
- Auto-categorization (AI)
- Auto-tagging: color, fabric, type, season
- Manual editing option

#### 2.2 Wardrobe Management
- Grid view by categories
- Search & filter
- Edit item details
- Delete/archive items

#### 2.3 Wardrobe Analytics
- Color palette visualization
- Cost-per-wear calculator
- "Haven't worn in 6 months" suggestions
- Duplicate detector
- Capsule wardrobe suggestions

---

### 3. Trip Planning
#### 3.1 Create Trip
- Destination search
- Dates selection
- Occasion tags (business, vacation, wedding, festival)
- Travel companions
- Cultural sensitivity toggle

#### 3.2 Weather Integration
- Real-time weather for destination
- 7-day forecast
- Auto-adjust recommendations based on temp/rain

#### 3.3 Day-by-Day Calendar
- Timeline view of trip
- Assign outfits to days
- Weather for each day displayed
- Outfit repeat warnings

#### 3.4 Packing List
- Auto-generated from selected outfits
- Checkbox functionality
- Add custom items
- Share as PDF/text
- Luggage weight estimator

#### 3.5 Laundry Planner
- Suggest when to wash items mid-trip
- Based on wear frequency

---

### 4. AI Stylist (Basic - Phase 1)
#### 4.1 Form-Based Input
- Destination
- Dates
- Occasion
- Number of outfits needed

#### 4.2 Recommendations
- Prioritize YOUR wardrobe first
- Suggest missing items to buy
- 3 outfit options per request
- Save/Reject feedback for learning

---

### 5. Notifications
- Trip reminders (3 days before)
- Weather change alerts
- Outfit suggestions for today
- Style challenge updates

---

---

### 7. AR Try-On
- Camera feed overlay
- Basic AR outfit preview
- Save/share AR photos

---

### 8. Social Features (Full Social Media)
#### 8.1 Friend System
- Send/accept friend requests
- Follow/unfollow users
- Friend suggestions based on style/trips
- Block/unblock users
- View friend list

#### 8.2 Profile Privacy
- **Public profile**: Anyone can view
- **Private profile**: Only approved followers can view
- Hide specific outfits/trips
- Hide wardrobe
- Activity privacy settings

#### 8.3 Sharing & Collaboration
- **Share outfits** with friends
- **Share trips** with friends
- **Collaborative trip planning**: Invite friends to plan together
- **Outfit recommendations from friends**: Suggest outfits to friends
- **Borrow/share wardrobe items** (track loans between friends)
- Group trips: Plan trips with multiple friends

#### 8.4 Social Feed
- Home feed with friends' posts
- Outfit posts with likes/comments/saves
- Trip posts with day-by-day outfits
- Repost/share friends' content
- Tag friends in posts
- Hashtags for discoverability

#### 8.5 Share Trip Looks
- Post outfit plans publicly
- Like/comment
- Share to Instagram

#### 8.6 Destination Style Feed
- See what others wore in same city
- Filter by date/occasion

#### 8.7 Outfit Voting
- Post 2 options
- Community votes

#### 8.8 Trip Twin Finder
- Find someone going to same destination same dates

#### 8.9 Creator Marketplace
- Curated outfit packs
- Buy/sell presets

#### 8.10 Direct Messages
- Chat with friends 1-on-1
- Group chats for trip planning
- Share outfits/trips via DM

---
